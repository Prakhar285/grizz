# Grizz

Grizz is a terminal-based productivity application built in Python. It started as a learning project to strengthen my understanding of Python, Object-Oriented Programming, Git, and software design. Instead of creating many disconnected beginner projects, I decided to build one project that will evolve as I learn new technologies throughout my AI & Data Science journey.

The current version focuses on core productivity features while maintaining a simple and modular codebase that can be extended in future versions.

---

## Current Features

### Task Manager

* Add new tasks
* View existing tasks
* Mark tasks as completed
* Persistent storage using JSON

### Notes

* Create notes with titles
* View saved notes
* Persistent storage using JSON

### Goals

* Module planned for upcoming versions
